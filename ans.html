Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that occurs on an element propagates upwards through its ancestors in the DOM tree, starting from the target element and moving up to the root of the document element.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements: document.createElement()
Adding Elements to the DOM: document.getElementByid().appendChild(newElement);
Removing Elements from the DOM: document.getElementByid().remove();
